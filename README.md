# DreamPortal – Kartografija sanj

Interaktivna aplikacija za mapiranje sanjskega sveta.

## Funkcije
- Vnos sanj
- Avtomatsko prepoznavanje lokacij
- Relativno pozicioniranje (sever = spodaj!)
- Drag & drop
- Plasti (zgornji/spodnji svet)
- Tranziti med lokacijami

## Kako uporabljati
1. Vnesi sanje
2. Klikni "Dodaj sanje"
3. Premikaj mehurčke z miško
4. Shrani z gumbom "Shrani"
